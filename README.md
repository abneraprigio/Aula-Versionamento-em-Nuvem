# 🚲 BikeShare SP - Gestão de Inventário (Sprint 3)

## 🏢 Contexto Profissional
Este projeto faz parte do desafio de desenvolvimento da **GNTEC Holding** para o sistema **BikeShare SP**. O objetivo desta Sprint foi evoluir o MVP (Produto Mínimo Viável), saindo de uma estrutura estática para um sistema funcional capaz de gerenciar o inventário de bicicletas em tempo real.
